## Section 3: Instructions for reproducing your results

Download the dataset from the Kaggle link located in the data folder. Next, open the "Colab Link" from the scripts folder and select File → Save a copy in Drive. Upload the IMDB Dataset.csv file to Colab so that it can be accessed within the notebook.

Once the data is loaded, go to Runtime → Change runtime type → Select T4 GPU. 

After completing these steps, you should be able to run the notebook cells sequentially without issues.
